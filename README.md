# My dotfiles

This directory contains the dotfiles for my system

## Requirements

Ensure you have the following installed on your system

### Git

```
pacman -S git
```

### Chezmoi

```
pacman -S chezmoi
```

## Installation

Use chezmoi to install the dotfiles

```
$ chezmoi init https://github.com/KSaiAkshit/dotfiles.git
```
